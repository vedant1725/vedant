# Vedant Portfolio

Live site: https://vedant1725.github.io/vedant

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

- GitHub repo: https://github.com/vedant1725/vedant
- GitHub Pages: Static export (next.config.mjs).
- Build: `npm run build` → out/ folder for Pages.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).
